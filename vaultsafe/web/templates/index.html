About VaultSafe

VaultSafe is primarily a command-line application designed for secure password management. This web server provides a GUI for users preferring a graphical interface.

Getting Started

  1. Initialize VaultSafe using the command-line interface (CLI): 
    $ vaultsafe init
    Follow the prompts to set up your master password and initialize the vault.
  2. Once initialized, you can securely manage your credentials using VaultSafe's CLI features.
  3. For GUI access, enter your master password at the login page.
